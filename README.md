Voice Assistant Project (JARVIS)

🚀 Overview
A Python-based desktop voice assistant named JARVIS that responds to voice commands and performs various tasks like searching information, opening websites, and providing real-time responses.

✨ Features
🎤 Voice activation (“Jarvis” wake word)
🔍 Wikipedia search
🌐 Open websites (Google, YouTube, etc.)
⏰ Time and date responses
▶️ Play YouTube videos
🔄 Multi-threading support for smooth performance
🛑 Interrupt handling
